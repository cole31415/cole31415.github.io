重学javascript(一):数组

发表于2020-11-17更新于2020-11-20字数统计3k阅读时长23分阅读次数

前言

近期在梳理后台系统开发中的知识点，发现大部分工作来自于Table的增删改查操作，而列表的数据又大多数是数组，故数组的操作比重尤为关键。为了更好更快的解决日常开发中数组的操作，重学了 javascript 数组的知识，记录下重学的学习笔记。

Array

数组是一个存储元素的线性集合(Collection)，元素可以通过索引来任意存取，索引通常是数字，用来计算元素之间存储位置的偏移量。

方法

了解数组的操作方法，我们就可以在开发中快速的使用数组方法解决数组的问题，下面就一个一个的方法去深入了解数组的方法。

Array.from()

从一个类似数组或可迭代对象(可以获取对象中的元素,如 Map 和 Set 等)创建一个新的、浅拷贝的数组实例。

Array.isArray()

用于确定传入的值是否是一个 Array。

Array.of()

用于将一个值转换为数组(创建一个具有可变数量参数的新数组实例，而不考虑参数的数量或类型)。

Array.of() 与 Array 构造函数 的区别在于处理整数参数(Array.of()总是返回参数值组成的数组):

concat()

用于合并两个或多个数组。此方法不会更改现有数组，而是返回一个新数组。

copyWithin()

在当前数组内部，将指定位置的成员复制到其他位置（会覆盖原有成员），然后返回当前数组。

entries()

返回一个新的 Array Iterator 对象(返回遍历的数组键/值对)。

keys()

返回一个新的 Array Iterator 对象(返回遍历的数组键名)。

values()

返回一个新的 Array Iterator 对象(返回遍历的数组键值)。

every()

判断数组中的所有元素是否都满足某个函数条件，返回一个布尔值。

fill()

用于给定值填充数组(给定值填充数组中起始索引到终止索引的全部元素，不包括终止索引)，生成新数组。

filter()

创建一个新数组, 筛选(过滤)满足某个函数条件的所有元素。

find()

用于找出第一个满足函数条件的数组元素的值(values)，否则返回 undefined。

findIndex()

用于找出第一个满足函数条件的数组元素的索引(index)，否则返回 -1。

flat()

用于返回一个扁平化(可指定的深度递归遍历数组)的新数组，默认扁平化值为 1。

flatMap()

用于返回一个扁平化映射(map()方法)的新数组，默认扁平化值为 1。

forEach()

对数组的每个元素遍历执行一次给定的函数条件。

includes()

用来判断一个数组是否包含一个指定的值，根据情况，如果包含则返回 true，否则返回 false。

indexOf()

返回在数组中可以找到一个给定元素的第一个索引，如果不存在，则返回-1。

join()

将一个数组（或一个类数组对象）的所有元素连接成一个字符串并返回这个字符串。如果数组只有一个项目，那么将返回该项目而不使用分隔符。

lastIndexOf()

返回指定元素在数组中的最后一个的索引，如果不存在则返回 -1。

map()

创建返回一个新数组(数组返回值:每个元素遍历调用特定函数条件后的返回值)。

push()

将一个或多个元素添加到数组的末尾，返回数组的新长度(添加的元素改变的是原数组，不会创建新数组)。

pop()

删除数组最后一个元素，返回数组的新长度(删除的元素改变的也是原数组，不会创建新数组)。

reduce()

对数组中的每个元素执行一个特定的**reducer 函数(升序执行)**，返回一个数组元素的特定的单值。

reduceRight()

对数组中的每个元素执行一个特定的reducer 函数(升序执行)(元素执行顺序从右到左)，返回一个数组元素的特定的单值。

reverse()

将数组的元素位置颠倒逆转，并返回数组。

shift()

删除数组第一个元素，返回数组的新长度(删除的元素改变的是原数组，不会创建新数组)。

unshift()

将一个或多个元素添加到数组的开头，返回数组的新长度(添加的元素改变的是原数组，不会创建新数组)。

slice()

返回一个新的数组对象(由 begin 和 end 决定的原数组的浅拷贝(包括 begin，不包括 end))，原始数组不会被改变。

some()

测试数组中的至少一个元素是否满足特定函数条件，返回一个布尔值。

sort()

对数组中的元素进行适当排序(默认升序)，然后返回排序后的数组。

splice()

通过删除或替换现有元素和/或在适当位置添加新元素来更改数组的内容。

toLocaleString()

返回一个字符串表示数组中的元素(数组中的元素自动调用 toString()方法)。

toString()

返回一个字符串，表示指定的数组及其元素(数组中的元素自动调用 toString()方法)。
