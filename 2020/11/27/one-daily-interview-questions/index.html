什么是虚拟 DOM(2020/11/28)

**虚拟 DOM(VDOM)**是真实 DOM 的内存表示，一种编程概念，一种模式。它会和真实的 DOM 同步，在更新 DOM 树时，虚拟 DOM 使用 diff 算法对比新旧 DOM 的变化，实现局部更新，节省了浏览器性能。
