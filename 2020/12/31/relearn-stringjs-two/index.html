重学Javascript(二):字符串

发表于2020-12-31更新于2021-01-06字数统计1.5k阅读时长12分阅读次数

前言

上次梳理了有关 Javascript 数组的知识点，这次我们继续来梳理字符串的知识点，快速熟悉并掌握字符串的相关属性与方法。

String

String 全局对象是一个用于字符串或一个字符序列的构造函数。

方法

了解字符串的方法，我们就可以在实际开发过程中快速的解决与字符串相关的问题，下面就一个一个的方法去深入了解字符串的方法。

chatAt()

返回指定位置(下标索引)的字符。

charCodeAt()

返回在指定的位置的字符的 Unicode 编码。

concat()

将一个或多个字符串与原字符串连接合并，形成一个新的字符串并返回。

includes()

用于判断一个字符串是否包含在另一个字符串中，根据情况返回 true 或 false。

indexOf()

返回指定的字符串位置(字符串中第一次出现的指定值的下标索引)，如果未找到该值，则返回-1。

match()

返回搜索一个或多个字符串匹配正则表达式的结果。

padEnd() || padStart()

用一个字符串填充当前字符串，返回填充后达到的指定长度字符串。

repeat()

构造并返回一个新字符串，该字符串包含被连接在一起的指定数量的字符串的副本。

replace()

返回一个由替换值替换部分或所有的模式匹配项后的新字符串，原字符串不会改变。

search()

返回搜索匹配项(正则表达式||字符串)的索引。

slice()

提取某个字符串的一部分，并返回一个新的字符串，且不会改动原字符串。

split()

将字符串分割为字符串数组。

startsWith()

用来判断当前字符串是否以另外一个给定的子字符串开头，并根据判断结果返回 true 或 false。

substring()

提取字符串中两个指定的索引号之间的字符。

toLocaleLowerCase()

把字符串转换为小写。

toLowerCase()

把字符串转换为小写，不会影响字符串本身的值。

toLocaleUpperCase()

把字符串转换为大写。

toUpperCase()

把字符串转换为大写，不会影响字符串本身的值。

trim()

删除字符串两端的空白字符串(空白字符串: 所有的空白字符以及所有行终止符字符)。
