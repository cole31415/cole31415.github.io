高效的Coding，前端开发人员值得一看的前端开发工具

发表于2020-09-25更新于2020-09-25字数统计2k阅读时长10分阅读次数

前言

看着我稚嫩的代码，今天我应该也可以过个快乐的儿童节吧，哈哈哈哈哈。既然是儿童节，那我就要给小伙伴们发发礼物，什么礼物呢？那就是小编在使用的前端开发工具，保证你会有个难忘的儿童节。好，下面就由我带大家一一的看看礼物吧。

工欲善其事必先利其器

对于一个前端开发人员来说，拥有一款适合自己的“武器”是非常重要的。vscode 就是这样一款优秀的前端开发编辑器，因为小编也在使用这个编辑器，像别的 webstorm、sublime text 等也是很棒的。由于小编目前在使用 vscode，所以给大家分享下我的编辑器常用的插件。让大家一起去学习打磨属于自己的前端开发“武器”。

语言包

Chinese (Simplified) Language Pack for Visual Studio Code: 这是 vscode 编辑器中文（简体）语言包，可以为 vscode 提供本地化中文界面。

界面美化

Agila Theme: 界面主题，这款主题我个人比较喜欢，界面配色看着舒服。小伙伴如果不喜欢，可以在插件库中寻找自己喜欢的主题进行设置。

vscode-icons: vscode 图标，这个插件可以很好的区分.css、.js 等文件图标，大大提高开发效率，强烈推荐大家使用。

代码美化

Bracket Pair Colorizer: 括号对色器，这个插件可以快捷的找到括号的另一半，简而言之就是能轻易的在代码海中找到左右括号。

Rainbow Brackets: 彩虹括号，为圆括号、方括号和大括号提供彩虹色，能在开发中快捷找到括号对。

indent-rainbow: 缩进彩虹，一个简单的扩展，促使代码的缩进更易读懂。

Syntax Highlighter: 语法突出显示，提供所有编程语言的通用语法着色引擎，凸显关键字、自定义函数等语法着色区分。

Prettier - Code formatter: 格式化美化代码，漂亮的代码书写总是每个程序员的追求，这个插件可以配置属于自己的代码格式，一键优化杂乱的代码。

便捷式编码

Auto Close Tag: 自动标签闭合，自动添加 HTML/XML 的标签闭合。

Auto Rename Tag: 自动标签重命名，自动重命名配对好的 HTML/XML 标签。

Auto Import: 自动导入，自动查找解析并提供所有可用可导入的代码操作。

Code Spell Checker: 代码拼写检查器，一个简单的代码拼写检查插件，可以快速的捕获在编码中的书写错误，保持高效的编码。

ES7 React/Redux/GraphQL/React-Native snippets: ES7/JS 代码段，这个插件可以使用简短的代码段书写繁琐的代码，提高开发效率。

React-Native/React/Redux snippets for es6/es7: React-Native/es6/es7 代码段，这个插件也可以快速的书写代码段，可以对比着使用。

Path Intellisense: 智能路径，快捷高效的引入组件的路径，节省了书写组件路径名的额外开发时间。

Git 功能插件

Azure Repos: Azure 存储库，它允许我们监视、构建、管理 Git 源存储库的拉取请求和更多的工作项。

Git Graph: Git 图，可以查看个人的存储库以及项目组库的 Git 库，可以轻松的进行合作开发。

GitLens — Git supercharged: GitLens，GitLens 增强了 vscode 中内置的 Git 功能，同时可以清晰的查看代码作者的身份等信息，可以促使项目组更好的去 Review 代码。

VSCode 检修插件

Fix VSCode Checksums: vscode 检验和修复器，用于检查 vscode 编辑器的校验和，应用校验和更改重启 vscode，消除编辑器核心文件警告。

众人拾柴火焰高

一个人的力量总是有限的，我们要学会与他人合作，学会交流沟通。小编觉得自己身处的项目组沟通合作就很棒，能快速的了解到项目的进度，能及时的掌握最新的需求变动等等。那我们项目组是如何做到这样默契的配合呢，接下来就由我给大家继续推荐儿童节礼物，合作沟通的方式。沟通是一个前端开发人员必备的一项技能，因为它可以让你的技术进一步提升，还可以提升个人的交流度。不管你是技术大牛还是技术小白都需要和别人沟通交流，能让他人清晰的了解到你的意图与需求。所以这就给大家发放一些小组合作的一些方案，能更清晰的发现问题，这样就可以及时的去沟通交流，促使项目更快的开发完毕。

石墨文档(强烈推荐)

石墨文档是小编目前用的最多的合作方式，不仅实时更新快，而且合作简便易懂。简单的列举项目中可以使用石墨文档建立的文档，具体文档可以分为下列几等:

  • 项目迭代文档: (确定项目开发时间、开发人员安排、项目开发需求原型图、测试用例等)

  • bugs 收集处理表: (测试人员收集测试过程中的 bugs，与开发人员同步进行，实行测试修复同步操作)

  • 项目组开发人员具体开发时间节点表: (安排开发人员的开发周期，具体的安排每一天的开发工作，明确开发进度)

合作开发

Worktile: worktile，可以使用 worktile 建立 bugs 处理文档，让开发人员及时的处理未解决的 bugs 和掌握已解决完毕的 bugs。

RAP2: rap2 接口说明文档，接口说明文档，让前后端开发及时了解联调的是哪个接口以及让前端开发人员了解接口返回数据格式等，促使前后端去沟通交流接口数据的统一性。

Leangoo 领歌: 可视化实时协作，让开发更透明、更高效。领歌有着灵活的看板模式，可以建立定义不同人员的工作流程，拖动任务体现开发人员工作进度，让团队成员之间的即时协作更好更快。

好马配好鞍，好船配好帆

一个人拥有了一个好的“武器”，同时也拥有一个完好的团队，可是这样还不够，我们个人也得加强自己的其他装备，就像主题说的一样，好马配好鞍，好船配好帆。没有人会嫌弃自己变得更好更优秀，下面就给大家强烈的推荐我额外的强力“装备”。

开发工具

Snipaste: 这是一款截图工具，小巧却功能强大，它不仅可以快速的截图，还可以集成标注和贴图功能，让截图可以实时固定在我们的可视区域，强烈推荐！！！

Microsoft To Do: 这是微软的一个个人任务、资料分享工具，可以实时的安排自己的工作进程、也可以分享自己的技术心得给小伙伴.

WinSCP: 这是一个 Windows 环境下使用的 SSH 的开源图形化 SFTP 客户端。同时支持 SCP 协议。它的主要功能是在本地与远程计算机间安全地复制文件，并且可以直接编辑文件。

spy-debugger: 微信调试，各种 WebView 样式调试、手机浏览器的页面真机调试。便捷的远程调试手机页面、抓包工具，支持：HTTP/HTTPS，无需 USB 连接设备。

PageSpeed Insights: PSI 优先分析网页的内容，然后提供关于如何提升网页加载速度的建议。强烈推荐！！！
